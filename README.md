# 🧠 AI-Based Smart Waste Segregation & Recycling Assistant

🚀 Developed during Internal Hackathon – Techies "N" Smartathon

---

## 📌 Problem Statement
Improper waste disposal leads to pollution, health risks, and inefficient smart city management.

---

## 💡 Solution
This project uses AI (Deep Learning) to:
- Detect waste type from images
- Classify into:
  - Plastic
  - Paper
  - Metal
  - Organic
- Suggest:
  - Proper disposal methods
  - Recycling tips
  - Environmental impact 🌍

---

## ⚙️ Tech Stack
- Python
- TensorFlow / Keras
- MobileNetV2
- Streamlit

---

## 🧪 Features
✅ Image upload  
✅ Waste classification  
✅ Disposal guidance  
✅ Environmental awareness  
✅ Beginner-friendly UI
